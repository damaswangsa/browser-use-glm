# Examples

## Post to Threads
```bash
./examples/post_to_threads.sh "Hello from AI agent! 🤖"
```

## Browse and extract anything
```bash
./examples/browse_and_extract.sh "Open https://news.ycombinator.com and get the top 5 posts"
```

## Get latest AI news
```bash
./examples/ai_news.sh
```

## Run directly with Python
```bash
source ~/browser-agent/bin/activate
python ~/browser_agent_runner.py "YOUR TASK IN NATURAL LANGUAGE"
```
